Anonymous Functions in 3 Languages

What are anonymous functions?

An anonymous function does not have a specific name that identifies it. Rather, it is passed around a program as a piece of data to be called at later times. Anonymous functions are very handy when you need to pass a unique function as an argument, as you can write the code for it right in the parameter list of the other function you're calling instead of having to break up your code. They can also be used to briefly create a new scope without having to put that code in a totally different place in your codebase.

How to make an anonymous functions in JavaScript

JavaScript syntax offers two ways to create an anonymous function:

1: using the function keyword:

// creating a new scope by creating a function and calling it immediately function() { // whatever your function needs to do }() // Or, passed as a parameter: setTimeout( function(){/* code to be run in 1500 seconds */}, 1500)

2: using arrow functions:

() => { // function body }()

Of course, these functions can accept parameters:

let desiredItems = array.filter( (item) => { return item > 1} )

In C#

C# also allows for two methods for generating anonymous functions:

1) using the delegate keyword:

List<Item> desiredItems = list.Where( delegate(Item item) { return item.Name.length > 0; } );

2) lambda expressions:

List<Item> desiredItems = list.Where( item => item.Name.length > 0 // the lambda expression );

In Python

Python has its own version of lambda expressions:

desired_items = list(map( lambda x: len(x.name) > 0 , items_list))

These lambda functions can actually take any number of arguments:

argless_func = lambda : "Doesn't do much" # argless_func() always just returns "Doesn't do much" three_arg_func = lambda a, b, c : a*b + c # three_arg_func(2,3,4) will return 10

In summary, that's five ways across three programming languages to write anonymous functions. See if you can use them to make your code a little more awesome!
