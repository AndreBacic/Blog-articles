HTML Setup

Firstly, the HTML file needs to include a link to the CSS file so the browser can apply the styles, and a meta tag to make elements and measurements the same across all devices:

<head> <link rel="stylesheet" href="css/style.css"> <meta name="viewport" content="width=device-width, initial-scale=1"> </head>

Because different devices have different pixel densities, CSS measurements (like px and rem) the meta tag is necessary to override the defaults.
